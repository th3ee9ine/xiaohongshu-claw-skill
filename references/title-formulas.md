# 小红书标题公式库

## ⚠️ 硬性限制
- 标题严格 ≤ 20 字（含标点符号）
- 禁止"最好/第一/唯一"等绝对词 → 会被判违规限流
- 禁止"震惊/惊天大秘密"等标题党

## 六大标题公式

### 1. 数字法（推荐）
- 示例：`住了 7 天发现的 5 个秘密`

### 2. 地点 + 结论法（推荐）
- 示例：`杭州｜这家酒店让我哭了`
- 示例：`上海探店｜巷子里的宝藏咖啡馆`

### 3. 情绪共鸣法
- 示例：`为什么我每次去杭州都要住这里`

### 4. 反差法
- 示例：`看起来便宜，但体验比五星还好`

### 5. 问句法
- 示例：`为什么博主都在囤这个？`

### 6. 悬念法
- 示例：`去了三次西湖，每次必做这一件事`

## 字数检查
```python
len("你的标题") <= 20  # 必须为 True
```
