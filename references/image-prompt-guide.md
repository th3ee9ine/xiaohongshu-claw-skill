# 小红书 AI 配图 Prompt 参考库

## 封面图 Prompt 模板

### 旅游打卡封面
```
Bright travel photo at [LOCATION]. [SCENE].
Warm golden hour light, scenic landmark, high-saturation, 3:4 vertical.
```

### 美食探店封面
```
Appetizing food photography of [DISH/PLACE].
Top-down flat lay, vibrant colors, bokeh background, food blog style, 3:4 vertical.
```

### 好物种草封面
```
Aesthetic product flat lay of [PRODUCT].
Pastel background, natural light, Pinterest-style, 3:4 vertical.
```

### 知识干货封面
```
Clean minimal desk setup with [THEME] stationery.
Pastel tones, soft natural window light, Muji style, 3:4 vertical.
```

## 正文配图

### 环境/场景图
```
Interior/exterior photo of [PLACE]. [ATMOSPHERE]. Natural light, 4:3.
```

### 细节图
```
Close-up detail shot of [SUBJECT]. Macro photography, bokeh, 4:3.
```

## Prompt 写作原则

1. **具体化**：给出场景、光线、色调、构图
2. **比例指定**：封面用 3:4，正文用 4:3 或 1:1
3. **色调统一**：同一篇笔记配图保持色调一致
4. **避免文字**：图片中不要要求生成中文文字
5. **配图说明**：图片下方统一标注 "AI 生成"
